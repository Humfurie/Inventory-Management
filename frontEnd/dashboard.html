Add new Product

Create

  1. Click the "Add New button"
  2. Fill in the input fields:
    • Product Name refers to the name of the product.
    • Product Description refers to the product's description or specifications about this product.
    • Unit price refers to the product's price.
    • Stocks Left refers to the number of stocks available.
  3. Click the "Submit" button to add the new product to database.

Edit

  1. Hover your mouse to the product you want to edit the product's information.
  2. Click the "Edit" button to enter edit product's information.
  3. Modify the information you want to change.
  4. Click "Update" button to update the product's information.

Delete

  1. Hover your mouse to the product you want to delete the product.
  2. Click the "Delete" button to delete the product from the database.
